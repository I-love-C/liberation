# Liberation

gRPC server connecting to Libre Office, the kit or the desktop in headless

## Install

Needs plugin for scala codegen in grpc, and needs protoc installed
```bash
wget https://github.com/scalapb/ScalaPB/releases/download/v1.0.0-alpha.3/protoc-gen-scala-1.0.0-alpha.3-linux-x86_64.zip
unzip protoc-gen-scala-1.0.0-alpha.3-linux-x86_64.zip
sudo mv protoc-gen-scala /usr/local/bin/
sudo chmod +x /usr/local/bin/protoc-gen-scala
```
